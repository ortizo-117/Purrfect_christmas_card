## Scene Setup

1. Choose a **quiet room** with minimal chaos.
2. Add subtle holiday elements:

   * A tree branch (secured)
   * Wrapped boxes (empty, obviously)
   * Neutral blanket or backdrop
3. Place cat **before** adding accessories.

> Pro tip: Let the cat explore the setup first. This gives them the illusion of control.
